# 기본적인 폴더 구조
```
your-repository/
│
├── _config.yml
├── _posts/
│   ├── example-post.md
├── _layouts/
│   └── default.html
├── index.md
└── other files...
```

## _config.yml
- Jekyll 블로그의 설정 파일로, 사이트 제목, URL, 레이아웃 등을 정의
    - Jekyll은 제킬이라고 읽으며, 지킬 앤 하이드에서 따왔다고 함. 근데 제킬이라고 읽는다고 함.
